# Spotify Clone (Full Stack)

A full-stack Spotify-inspired music streaming application built using the MERN stack.

## Features
- User Authentication
- Music Streaming
- Playlist Management
- Responsive UI
- Secure Backend APIs

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Installation

### Backend
cd backend
npm install
npm start

### Frontend
cd spotify-frontend
npm install
npm run dev
